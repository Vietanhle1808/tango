Tango template is based on Bootstrap v3.3.5 - a collection of CSS and JS files that can help you rapidly develop sites that look beautiful at any size, be it a 17" laptop screen or an iPhone.

Bootstrap's base grid is a variation of the grid system. The syntax is simple and it's effective cross browser, but the awesome part is that it also has the flexibility to go mobile like a champ. You can really easily customize all part of site or build new.

The default Bootstrap3 grid system is composed by 12 columns, named like "col-*-1", "col-*-2", "col-*-3"...."col-*-12". Grid systems are used for creating page layouts through a series of rows and columns that house your content.

Header Image Parallax:

header-parallax made for making header image parallax. This Div contains 2 data attributes.

  • data-parallax-background-image
  • data-stellar-background-ratio

 

header-parallax: has this starting following code:

<section class="header-featured-content">
    <div class="header-parallax" data-stellar-background-ratio="0.5" data-parallax-background-image="image3.jpg">

    </div><!-- end header-parallax -->
</section><!-- end header-featured-content -->

 

- You can specify custom Background Image using data attribute called data-parallax-background-image where you can give it value of image which must be existed in images/files/parallax-background-images  folder.

- To make Parallax Effect on the background image, add data attribute called data-stellar-background-ratio which add this effect with proper value. The recommended value might be "0.5". This effect based on "Stellar" Jquery Plugin.

It looks like this:

<div class="header-parallax" data-stellar-background-ratio="0.5" data-parallax-background-image="image3.jpg">

For Colored Layer Background, i added 2 custom options, Background Color and Background Color Opacity. These 2 options are not specialized for this layer, they can be used in any place around the template. They will be mentioned in the "Styling Options" section.

Both of these options don't have default values. They are as following:

  • Background Color : Add data-background-color.
    It must have 6 letters of Hex value.
  • Background Color Opacity : Add data-background-color-opacity.
    You can change the opacity value by adding value between 0 to 1.

It looks like this:

<div class="color-overlay" data-background-color="#000000" data-background-color-opacity="0.75"></div><!-- end color-overlay -->

General Data Attribute Options

General Data Attribute Options

The use of Data Attribute Options gives the ability to make custom and quick styling without making custom classes or ids.

These options are avialable for almost all general content styling features like Background, Divider Space.

These data attribute options provide specific and direct customization into HTML if you want prefer this without making custom class or id.

 

They will be handled as following:

  • Background
  • Divider Space

 

Background

Basic background options are included. They are as following:

  • Background Color : Add data-background-color.
    It must have 6 letters of Hex value.
  • Background Color Opacity : Add data-background-color-opacity.
    You can change the opacity value by adding value between 0 to 1.
  • Parallax Background Image : Add data-parallax-background-image.
    Where you can give it value of image which must be existed in images/files/parallax-background-images folder.
<div data-parallax-background-image="image1.jpg">
<div data-background-color="#000000" data-background-color-opacity="0.3">
Divider Space

Sometimes you need to add specific space between contents to make contents look well organized. You can do this by adding div that you can set its height manually between contents. Use div with class "divider-space" and data attribute data-divider-space with value of height you want.

Here is how Divider Space code looks:

<div class="divider-space" data-divider-space="25px"></div>

This custom features add the ability to add individual style option for some elements that's used repeatly like Overlay Pattern and Colored Layer Background and more features that i will talk about it in a main section that will be called "Styling Options". Almost all of these features based on custom data attribute.

For Overlay Pattern, i added 2 custom options, Pattern Overlay Background Image and Pattern Overlay Darkness Opacity. Both of them have default values "Yes" and "0.8" respectively. They are as following:

  1. Pattern Overlay Background Image : Add data-pattern-overlay-background-image.
    To disable using the background image pattern, add value "none".
  2. Pattern Overlay Darkness Opacity : Add data-pattern-overlay-darkness-opacity.
    You can change the darkness value by adding value between 0 to 1.
<div class="pattern-overlay" data-pattern-overlay-darkness-opacity="0.85" data-pattern-overlay-background-image="none"></div>
